# Gestão de Pedidos - EGITRON

Desafio Técnico Backend (Spring Boot + Java 8 + MSSQL)

## Tecnologias
- Java 8 (Zulu)
- Spring Boot 2.7.18
- Maven
- Hibernate / JPA
- SQL Server
- Postman (testes de API)
- Mailtrap (testes de e-mail)

## Requisitos
- JDK 1.8 instalado
- Maven 3.8+ instalado
- SQL Server em execução

## Como correr o projeto
```bash
mvn spring-boot:run
